Fiat documentation
=========================

Fiat is a library for managing projects, often times deploying
those projects on one or more other machines. Fiat attempts to
provide a best practices workflow for all stages of software
engineering.

Fiat provides version control, a makefile-esque build process,
automated testing, documentation, and deploying, configuration
management, application management, and more.
